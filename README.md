This Power BI project provides a **key operational and analytical view** of hospital management services, focusing on patient experience and resource utilization.

### **Key Performance Indicators (KPIs)**

The dashboard tracks four main KPIs:

* **Total Patients:** 1,000
* **Avg. Satisfaction:** 79.60
* **Avg. Stay (Days):** 7.41
* **Total Discharged:** 1 Million

### **Core Insights**

| Visualization | Focus | Key Takeaway |
| :--- | :--- | :--- |
| **Service Utilization** | Shows patient volume across **Emergency, Surgery, General Medicine, and ICU**. | **ICU** has the highest volume, contributing to the 1M total. |
| **Length of Stay** | Analyzes average stay by age group. | **Age 40+** group accounts for the longest average stays (approx. 4K units). |
| **Satisfaction Levels** | Compares satisfaction scores by department (Emergency, ICU, etc.). | Identifies specific departments needing **quality improvement efforts**. |
| **Trends** | Charts **Admission and Discharge** volumes over time (Jan-Jul 2025). | Allows for **capacity planning** and monitoring operational flow. |

### **Project Details**

* **Tool:** Microsoft Power BI Desktop
* **Data Source:** Simulated/Dummy Hospital Operational Data
* **Purpose:** To enable **data-driven decision-making** for resource allocation and service optimization.
